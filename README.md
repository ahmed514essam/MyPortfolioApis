## 🔹 Project Overview
This project is a **professional Web API** built with **ASP.NET Core**, demonstrating modern backend development skills.  
It highlights **secure authentication using JWT** and **image/file management via Cloudinary**, making it suitable for integration with frontend applications or portfolio showcase.  

---

## ⚡ Key Features
- **Secure Authentication:** Users login/register using JWT tokens for protected access to API endpoints.  
- **Cloud Media Management:** Upload, store, and manage images with Cloudinary, while keeping links in the database.  
- **RESTful API Endpoints:** Well-structured endpoints following REST principles.  
- **Extensible Architecture:** The project structure allows easy addition of new features.  

---

## 🎯 Project Purpose
The purpose of this project is to demonstrate my ability to build **robust and secure backend APIs**, focusing on:  
1. **Data Security** – Sensitive operations protected via JWT.  
2. **Media Handling** – Cloudinary integration for scalable media management.  
3. **Professional Practices** – RESTful design and organized code structure.  

---

## 🛠️ Technologies
- **Backend:** .NET 7 / ASP.NET Core  
- **Authentication:** JWT (JSON Web Tokens)  
- **Cloud Storage:** Cloudinary  
- **Database:** SQL Server  
- **Development Tools:** Visual Studio 2022, Postman
